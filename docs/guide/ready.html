  • master 演示源码，即在框架源码基础上提供各种特性展示，含有大量示例代码
  • template 框架源码，不含示例代码，适合直接用于项目开发
  • 手动下载

# 开发环境

使用本模板前，需要在本地依次安装好 Node.jsopen in new window, pnpmopen in new window, Gitopen in new window 和 Visual Studio Codeopen in new window，并且在 Visual Studio Code 里安装好以下扩展：

  • EditorConfig for VS Codeopen in new window
  • DotENVopen in new window
  • ESLintopen in new window
  • stylelintopen in new window
  • Vue Language Features (Volar)open in new window

TIP

以上 5 个为 Visual Studio Code 必备扩展，以下则是作者建议安装的扩展，安装它们将在一定程度上提升开发效率。

  • Chinese (Simplified) Language Pack for Visual Studio Codeopen in new window 中文语言包
  • Auto Close Tagopen in new window 自动添加 HTML 的闭合标签
  • Auto Rename Tagopen in new window 自动重命名配对 HTML 标签
  • Color Highlightopen in new window 在代码中高亮颜色
  • Color Pickeropen in new window 取色器
  • Highlight Matching Tagopen in new window 高亮显示匹配的标签
  • Image previewopen in new window 图片预览
  • indent-rainbowopen in new window 彩虹缩进提示
  • Path Intellisenseopen in new window 路径智能提示

除此之外，还需要准备好开发使用的浏览器，推荐使用 Chromeopen in new window ，并且在 Chrome 里安装好 Vue.js devtoolsopen in new window 扩展，这一步很重要，安装 Vue 官方提供的调试工具将大大提升解决 bug 的效率。

不过由于安装 Vue.js devtools 需要访问 Chrome 应用商店，如果在大陆地区访问，需要自行准备相关可访问外网的工具。如果不具备该条件，也可访问 Vue.js devtoolsopen in new window 官网了解本地构建或者下载支持 Firefox 的 Vue.js devtools 。
