# Chip 纸片 <Badge type="tip" text="专业版" vertical="top" />

## Props

| 参数     | 说明             | 类型    | 可选值                                      | 默认值 |
| :------- | :--------------- | :------ | :------------------------------------------ | :----- |
| type     | 类型             | string  | primary / success / warning / danger / info | -      |
| closable | 是否显示关闭按钮 | boolean | -                                           | false  |

## Event

| 事件名称 | 说明                                         | 回调参数 |
| :------- | :------------------------------------------- | :------- |
| close    | 在点击由 `closable` 属性生成的清空按钮时触发 | -        |
