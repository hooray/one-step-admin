# SvgIcon SVG图标

## Props

| 参数   | 说明                                                                                         | 类型   | 可选值                       | 默认值 |
| :----- | :------------------------------------------------------------------------------------------- | :----- | :--------------------------- | :----- |
| name   | svg 文件名，文件存放在 `/src/assets/icons` 目录下，专业版支持设置 Element Icon 和 RemixIcon | string | -                            | -      |
| flip   | 翻转方式，支持水平、垂直以及同时翻转                                                         | string | horizontal / vertical / both | -      |
| rotate | 旋转角度                                                                                     | number | 0 - 360                      | 0      |
